## 4. 작업 중 해결한 오류

### Chrome 확장 파일 업로드 `Not allowed`

- 증상: 파일 선택 창은 열리지만 `.ait`를 전달할 때 `Not allowed`.
- 원인: ChatGPT Chrome Extension의 `Allow access to file URLs` 권한이 꺼져 있었음.
- 해결: `chrome://extensions` → ChatGPT Chrome Extension → Details → 해당 권한 활성화.
- 주의: 권한을 바꾸면 확장이 재시작되어 기존 브라우저 연결 ID가 무효가 될 수 있다. 새 Chrome 연결을 잡은 뒤 콘솔 탭을 다시 claim하고 업로드해야 한다.

### 검토 요청 버튼 비활성

- 증상: 서버 빌드는 `검토 필요`인데 검토 요청 버튼이 비활성.
- 원인: 공식 출시 절차상 **해당 새 버전의 테스트를 1회 이상 완료해야** 활성화됨.
- 해결: `테스트`에서 새 버전 QR/딥링크를 열고 사용자가 토스 앱에서 실제 실행.

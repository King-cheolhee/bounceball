## 1. 이번에 한 일

### 게임용 내비게이션

- `app/granite.config.ts`: `webViewProps.type: 'game'` 추가.
- 결과: 흰색 비게임 바, 뒤로가기, 로고, 앱 이름을 없애고 게임용 더보기·X만 우측 상단에 표시.
- 메뉴·설정의 중복 `✕ 닫기` 버튼 제거. 네이티브 X가 전 화면 종료 경로를 담당.

### iPhone·Android 우상단 충돌 방지

- `variables.css`: `--game-nav-reserved-right: 112px`, `--game-nav-reserved-height: 64px` 공통 정의.
- 메뉴·설정 상단 행에 공통 우측 예약 영역 적용.
- 플레이 HUD의 오른쪽 끝을 예약 영역 왼쪽으로 이동.
- 플레이 우측 상단 예약 영역에서 좌우 이동 입력이 발생하지 않도록 입력 차단 레이어 추가.

### SDK 갱신

- `@apps-in-toss/web-framework` 2.7.0에서 2.10.5로 갱신.
- 이유: 2.7.0 게임용 X는 즉시 `closeView()`였지만 2.10.5는 프레임워크 종료 확인 모달을 먼저 표시한다. 게임 출시 가이드의 종료 확인 요건 대응.
- 새 패키지는 추가하지 않았고 기존 공식 SDK만 갱신했다.

## 2. ⚠️ 알아둘 것
- **TG-OM-26-00588은 절대 철회 금지.** 옛 초안 TG-OM-26-00502("작성중", 6/16 미제출분)가 목록에 남아 있음 — 무해, 원하면 그것만 철회 가능.
- 카드결제 취소는 회의 개최 1일 전까지, 거부예정 결정 후엔 수수료 환불 불가.
- **GRAC 사이트 자동화 요령**(재사용): NICE 인증 팝업은 자동화 브라우저에서 opener 단절 → 부모창에서 `setMobileCert('ok')`로 복구. DEXT5 업로드는 `DEXT5UPLOAD.Transfer('upload1')`. 간소화 설문(scs.grac.or.kr)은 크로스오리진 iframe이라 클릭 무시 → `scs.grac.or.kr/step2?applicationId=N` 별도 탭으로 열어 제어, 완료 후 부모창에서 `window.postMessage({result:'success', id:'<sbmtId>'}, '*')`. 상세는 메모리 [[tangtangball-release-facts]].
- 등급분류/사후관리는 **게임물내용설명서 기준**으로 진행됨 — 게임 내용(기믹·과금·표현)을 바꾸면 내용수정신고 필요할 수 있음. **출시 전 게임 내용 변경은 신중히.**
- 재배포 필요시: `cd app && npx vite build --base=/bounceball/ --outDir dist-pages` → gh-pages 브랜치 갱신(worktree 방식, session_v3·메모리 참조).

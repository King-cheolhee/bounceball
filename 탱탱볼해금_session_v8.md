## 4. 주의사항
- **검토요청·출시는 비가역 → 사용자 직접 클릭.** AI는 임의 클릭 금지.
- **콘솔 세션 ~1시간 만료**: 저장이 HTTP 200 + `resultType:"FAIL"`(4010)로 위장. 응답 본문 확인. 재로그인 사용자 직접.
- **등급분류 방식은 "게임물 등급분류증명서"**여야 함(스토어 링크 아님 — 앱인토스 전용이라 스토어 링크 없음).
- **스크린샷은 세로 전용**(가로 넣지 말 것). 게임 챕터명 영문(LCD FIELD 등)·게임 HUD(STAGE/EXIT)는 제명 아니라 유지 OK.
- 실광고 ID 교체 시 재빌드·재등록 필요(재심사 대상일 수 있으니 타이밍 주의).
